# 获取授权码

> 联系管理员购买授权码

# 授权码配置

购买后，在每个 heki 实例的配置中加入以下配置，并重启 heki：

```
heki_key=授权码
```

# 授权对接地址

每个授权码绑定一个对接地址，例如 `panel_url` 对应的地址。

对接地址将在第一次使用授权码时自动绑定，绑定后，该授权码不可使用其它对接地址。

# 限制人数说明

人数只计算包含有效套餐、未到期、流量未超出、未封禁的用户；即`能正常使用代理的用户`。

例如：网站里总共注册了 1000 名用户，其中 800 名用户已购买套餐且未到期、流量未超；剩余 200 名用户为试用套餐，其中 100 名用户正在试用 (未到期、流量未超)，剩下 100 名试用已到期或试用流量已超出；此时该网站有效人数为 900 人

# 在线授权

1. 支持升级套餐（增加人数或后端类型），根据剩余时间补差价
2. 人数超出会掉线，记得提前升级
3. 购买后获取一个`授权码`，该授权码绑定一个`对接地址`
